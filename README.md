# 🌱 Crop Recommendation System using Machine Learning

## 📌 Description
This project is a machine learning-based application that recommends the most suitable crop to grow based on environmental and soil conditions.

It helps farmers make better decisions by analyzing key parameters such as temperature, humidity, and soil nutrients.

---

## 🚀 Features
- Predicts suitable crops based on input data  
- Uses machine learning algorithms for better accuracy  
- Simple and easy-to-use interface  
- Data-driven decision support system  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  

---

## 📂 Project Structure
- `Crop_Recommendation.py` – Main model logic  
- `Labeled_Data.csv` – Dataset used for training  
- `static/` – Static files (CSS, JS)  
- `templates/` – HTML templates  

---

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/kdevaraj9177-cyber/crop-recommender.git

2. Navigate to the project folder:
cd crop-recommender

3. Install required libraries:
pip install pandas numpy scikit-learn

4. Run the program:
python Crop_Recommendation.py
